# Development Stack

## Purpose
Curated local stack for code development work, based on the Awesome OpenClaw Skills coding-related categories and the tools already available on this machine.

## Ready Local Skills
- `github-local` — GitHub repos, PRs, issues, Actions via `gh`
- `byterover-local` — lightweight repo/code memory
- `gemini-cli-local` — local Gemini CLI coding/reasoning workflow
- `agent-builder-local` — design/improve agents
- `agents-manager-local` — manage multiple agents/sessions
- `clawddocs-local` — OpenClaw docs/commands/config lookup
- `mcp-client-local` — MCP tool/server planning
- `automation-workflows-local` — repeatable local dev automations
- `filesystem-local` — safe project file operations
- `document-rag-local` — retrieve context from local docs/code-adjacent notes
- `markdown-converter-local` — convert docs/web pages into Markdown notes
- `self-improving-local` — capture mistakes and better workflows
- `skill-search-local` / `local-skill-catalog` / `skill-vetter-local` — skill discovery, mapping, and vetting
