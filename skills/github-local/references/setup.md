# GitHub local setup

## Preferred tool
- GitHub CLI: `gh`

## Checks
- `gh --version`
- `gh auth status`

## Local machine note
- Portable `gh` is available at `workspace/tools/gh/bin/gh.exe`

## If missing
Install GitHub CLI or use browser fallback.

## If unauthenticated
Use `gh auth login` or browser-based sign-in.
