# Vetting checklist details

## Questions to answer
- What exact problem does the skill solve?
- Does the skill require secrets or account auth?
- Does it write outside the workspace or message external services?
- Does it create cron jobs or persistent automation?
- Does it download/run extra code?
- Is there a simpler local alternative?

## Output format
- Summary
- Trust signals
- Risk signals
- Required secrets/auth
- Recommendation
- Safer alternative
