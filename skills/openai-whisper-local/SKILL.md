# OpenAI Whisper Local

Use local transcription when audio/video should be processed on-machine.

## Workflow
1. Check whether a Whisper-compatible CLI/runtime is installed.
2. Check for Python/runtime prerequisites if needed.
3. Prefer local transcription for privacy-sensitive audio.
4. Store outputs as text files in the workspace when appropriate.

Read `references/setup.md`.
