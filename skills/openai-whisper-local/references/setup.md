# Whisper local setup

## Typical requirements
- local Whisper CLI/runtime
- model files
- FFmpeg for media handling

## Local machine note
- Portable FFmpeg is available at `workspace/tools/ffmpeg/ffmpeg-8.0.1-essentials_build/bin/ffmpeg.exe`
- Portable Python is available at `workspace/tools/python/python.exe`

## Use cases
- audio transcription
- video transcript extraction
- meeting note generation
