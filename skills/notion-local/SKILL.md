# Notion Local

## Preconditions
- Requires Notion integration credentials/token.

## Workflow
1. Check whether a Notion client/CLI or local integration exists.
2. Check whether the token is configured.
3. Use structured API workflows when possible.
4. Fall back to browser workflows when API access is unavailable.
