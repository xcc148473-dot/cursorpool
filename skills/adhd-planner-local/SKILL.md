# ADHD Planner Local

## Workflow
1. Reduce scope to a manageable horizon.
2. Break work into small visible actions.
3. Prefer short task batches over long abstract lists.
4. Emphasize momentum, clarity, and restartability.
5. Avoid overloading the plan.
