# Task type mapping

## Browser relay
- logged-in web apps
- dashboards
- service configuration pages
- uploads/downloads in browser

## agent-browser CLI
- isolated web automation
- reproducible browser scripts
- snapshots/screenshots/structured extraction

## Shell/PowerShell
- file operations
- installs
- service control
- git, build, package, conversion tasks

## AutoHotkey / GUI fallback
- native Windows apps
- file dialogs
- installers
- legacy software
