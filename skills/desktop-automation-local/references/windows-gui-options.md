# Windows GUI automation options

## AutoHotkey v2
Best for keyboard shortcuts, hotkeys, window activation, simple clicking/typing, and light app automation.

## PowerShell UI Automation / .NET UIA
Best when using Windows accessibility tree and enterprise apps that expose automation properties.

## WinAppDriver
Best for structured Windows app automation in test-like flows.

## OCR / image matching
Use only as last resort.

## Recommendation
For this machine, start with AutoHotkey v2 for general GUI fallback if browser/shell are insufficient.
