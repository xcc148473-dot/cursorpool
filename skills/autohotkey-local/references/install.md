# AutoHotkey local install notes

## Target
- AutoHotkey v2

## Check
- `AutoHotkey.exe` or `AutoHotkey64.exe` on PATH
- common install locations under `C:\Program Files\AutoHotkey\`

## If installed
Record the exact executable path and prefer explicit invocation in scripts.

## If missing
Install AutoHotkey v2 before attempting native GUI automation.

## Recommendation
Use `.ahk` files with explicit v2 syntax and keep one script per task.
