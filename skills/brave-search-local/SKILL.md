# Brave Search Local

## Preconditions
- Requires a Brave Search API key.

## Workflow
1. Check whether the API key is configured.
2. If missing, document the requirement and use another search path.
3. If present, use Brave Search for quick retrieval and source-backed summaries.

Read `references/setup.md` for required configuration.
