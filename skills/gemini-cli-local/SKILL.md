# Gemini CLI Local

## Workflow
1. Check whether Gemini CLI is installed.
2. Check auth/login state.
3. Use it for local coding, reasoning, repo work, and CLI-driven tasks.
4. Fall back to the main OpenClaw/browser workflow if auth is incomplete.

Read `references/setup.md`.
