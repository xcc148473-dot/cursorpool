# Memory Manager Local

Use local files as the source of continuity.

## Workflow
1. Review recent daily memory notes.
2. Extract durable facts, decisions, preferences, and lessons.
3. Promote stable items into `MEMORY.md` when appropriate.
4. Keep daily notes raw and long-term memory curated.
5. Remove stale or incorrect long-term entries when replaced.
