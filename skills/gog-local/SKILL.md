# Gog Local

## Workflow
1. Check whether the Google Workspace CLI is installed.
2. Check auth status.
3. Use the CLI for mail, calendar, drive, contacts, sheets, and docs tasks.
4. Fall back to browser workflows if CLI auth/setup is incomplete.

Read `references/setup.md` before first-time use.
