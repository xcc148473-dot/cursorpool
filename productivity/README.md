# Productivity System

这是这台 OpenClaw 机器上的本地生产力系统总入口。

## 1. 今天要做什么
优先看这些：
- `../skills/daily-planner-local/`
- `../skills/priority-sorter-local/`

- 排优先级

- 把杂乱事项变成可做动作
